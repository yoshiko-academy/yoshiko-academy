ℹ 결제가 완료되지 않은 학생은 ✅ 결제완료 버튼을 누를 때까지 계속 표시돼요.
ℹ 결제완료 버튼을 누른 학생들의 이력이에요. 학생 목록에서 이번 달 결제 상태를 색으로 확인할 수 있어요.
ℹ 개인 수업 · 전화 수업 · 출강 학생만 표시돼요. 날짜를 클릭하면 출석 체크되고 캘린더에 자동으로 수업이 등록돼요. 파란 점(●)이 있으면 캘린더 연동 완료예요. 출강은 결제 알림이 없어요.
ℹ 캘린더에 등록된 수업 기준으로 계산돼요. 월을 변경하면 해당 월 수업으로 자동 전환돼요.
💡 반에 요일·시간을 설정하면 스케줄이 자동 생성되고, 반을 삭제하면 관련 스케줄도 자동 삭제돼요.

📱 폰에서 열기

가장 쉬운 방법 — GitHub Pages (무료)
  1. github.com 에서 무료 계정 만들기
  2. 새 Repository 만들기 (이름 예: yoshiko-academy)
  3. 이 HTML 파일을 index.html 로 이름 바꿔서 업로드
  4. Settings → Pages → Branch: main → Save
  5. https://계정명.github.io/yoshiko-academy 주소 생성!
  6. 폰에서 그 주소 열고 "홈 화면에 추가" 탭 → 앱처럼 설치 완료 🎉
💡 선생님 전용 출결 링크도 같은 주소에 ?mode=teacher&teacher=이름 붙이면 돼요!
